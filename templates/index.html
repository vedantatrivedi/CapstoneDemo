Image Dehazer Application

A Flask Application backed by Artificial Intelligence
for automatic single image dehazing.

Let's Get Started

Know our application

How does this app work?

An image can be uploaded on the web app. A call to our Flask API is made .

We use our trained GAN model to call the generate function on this image, and returns the dehazed image.Flask is used as the backend, with HTML/CSS for a responsive front end application.

Dehazer

Dataset

NYU-v2 dataset for clean and hazed images.

Method

An attention based Generational Adverserial Network. A computer Vision project.

Result

A web application which returns a dehazed image.
